# カーセンサー スクレイピング 簡単な使い方

## 🚀 最も簡単な方法（既知のURLがある場合）

### ステップ1: URLからコードを確認

カーセンサーのサイトで車種を選択して、URLを確認します。

例: `https://www.carsensor.net/usedcar/bBM/s033/f002/index.html`
- `bBM` = BMW（メーカーコード: BM）
- `s033` = M3セダン（車種コード: 033）
- `f002` = 2014年02月～2018年12月モデル（モデル年式コード: 002）

### ステップ2: Pythonでスクレイピング

```python
from carsensor_url_pattern_scraper import CarsensorURLPatternScraper

# スクレイパーを作成
scraper = CarsensorURLPatternScraper()

# データを取得
df = scraper.scrape_by_url_pattern(
    maker_code='BM',      # BMW
    model_code='033',     # M3セダン
    grade_code='002',     # 2014年02月～2018年12月モデル
    max_cars=31           # 最大31台
)

# CSVに保存
df.to_csv('result.csv', index=False, encoding='utf-8-sig')
print(f"✅ {len(df)}台のデータを取得しました")
```

## 📋 グレード一覧を自動取得する方法

### ステップ1: URLからグレードを取得

```python
from carsensor_grade_extractor import CarsensorGradeExtractor

extractor = CarsensorGradeExtractor()

# URLからグレード一覧を取得
url = "https://www.carsensor.net/usedcar/bBM/s033/f002/index.html?ROUTEID=edge"
grades = extractor.get_grades_from_url(url)

# 結果を表示
for grade in grades:
    print(f"- {grade['name']}")

# 保存
extractor.save_grades(grades, 'grades.json')
```

## 📚 辞書を作成する方法（トヨタ アルファード・ヴェルファイアなど）

### ステップ1: コードを確認

1. カーセンサーのサイトを開く
2. トヨタを選択: `https://www.carsensor.net/usedcar/bTO/index.html?ROUTEID=edge`
3. アルファードのリンクをクリックしてURLを確認
   - 例: `/usedcar/bTO/s123/` → 車種コードは `123`
4. モデル年式（40系など）を選択してURLを確認
   - 例: `/usedcar/bTO/s123/f001/` → モデル年式コードは `001`

### ステップ2: 辞書を作成

```python
from carsensor_dictionary_manual import CarsensorDictionaryManager

manager = CarsensorDictionaryManager()
manager.dictionary = {
    'maker_code': 'TO',
    'maker_name': 'トヨタ',
    'models': {}
}

# アルファードを追加（実際のコードを入力）
manager.add_model('TO', 'アルファード', '123')  # 実際のコード

# 40系を追加
manager.add_model_year('123', '40系', '001')  # 実際のコード

# グレードを自動取得
manager.auto_fill_grades('TO', '123', '001')

# 辞書を保存
manager.save_dictionary('toyota_alphard.json')
```

### ステップ3: 辞書を使ってスクレイピング

```python
# 辞書を読み込み
manager.load_dictionary('toyota_alphard.json')

# 全グレードをスクレイピング
combined_df = manager.scrape_all_from_dictionary(
    output_dir='output/toyota',
    max_cars_per_url=100
)
```

## 📝 実践例

### 例1: BMW M3セダンのデータを取得

```python
from carsensor_url_pattern_scraper import CarsensorURLPatternScraper

scraper = CarsensorURLPatternScraper()
df = scraper.scrape_by_url_pattern('BM', '033', '002', max_cars=31)
df.to_csv('bmw_m3.csv', index=False, encoding='utf-8-sig')
```

### 例2: トヨタ アルファードの全グレードを取得

```python
from carsensor_dictionary_manual import CarsensorDictionaryManager

manager = CarsensorDictionaryManager()
manager.dictionary = {'maker_code': 'TO', 'maker_name': 'トヨタ', 'models': {}}

# コードを確認してから実行
manager.add_model('TO', 'アルファード', '実際のコード')
manager.add_model_year('実際のコード', '40系', '実際のコード')
manager.auto_fill_grades('TO', '実際のコード', '実際のコード')
manager.save_dictionary('alphard.json')
```

## ⚠️ 注意事項

1. **コードの確認**: 実際のカーセンサーのサイトでコードを確認してください
2. **サーバー負荷**: 大量のスクレイピングは避け、適切な待機時間を設けてください
3. **利用規約**: サイトの利用規約を確認してください

## 🔍 コードの見つけ方（詳細）

1. **メーカーコード**: URLの `/b{CODE}/` 部分
   - トヨタ: `TO`
   - BMW: `BM`
   - メルセデスベンツ: `MB`

2. **車種コード**: URLの `/s{CODE}/` 部分
   - カーセンサーのサイトで車種を選択してURLを確認

3. **モデル年式コード**: URLの `/f{CODE}/` 部分
   - カーセンサーのサイトでモデル年式を選択してURLを確認
